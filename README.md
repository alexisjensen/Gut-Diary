# Gut Diary

A tiny browser-based gut symptom diary for tracking meals, bloating, nausea, cramping, burping, distension and bowel movements.

## How to use

Open `index.html` in a browser, or publish this folder with GitHub Pages.

Entries are saved in your browser with `localStorage`. Nothing is sent anywhere.

## GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js` and `README.md`.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.
6. GitHub will give you a web link for the app.

## Exporting

Use **Export CSV** for a spreadsheet or to send to a clinician. Use **Export JSON** for a backup.
